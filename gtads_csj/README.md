# 使用

### pubspec.yaml
```dart
 gtads_csj: ^0.0.2
```

### 引入
```dart
import 'package:gtads_csj/gtads_csj.dart';
```

### 使用
```dart
GTAds.addProvider(GTAdsCsjProvider(alias: "csj",androidId: "5098580",iosId: "5098580"));
```
